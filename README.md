# Project Overview
Derm AI is a skincare management mobile app that helps users navigate the complex world of dermatology. It provides diagnoses and targeted product recommendations along with other features like progress tracking and routine maintenance. 

# Tech Stack
Frontend: React Native
Backend: Firebase
ML/AI Models: GPT, Gemini
